    Image Compression

    Developed a Python-based lossless image compression tool using Run-Length Encoding (RLE) and Huffman coding. The system effectively reduces image file sizes without any loss of quality. RLE compresses repetitive pixel sequences, while Huffman coding optimizes the encoding of image data, resulting in a highly efficient compression process. This approach ensures the original image can be perfectly reconstructed, maintaining integrity while significantly reducing storage requirements.
